# Dependencies on brand new device

# Windows

- Go
    - Winget: winget install --id GoLang.Go -e --source winget
    - Website: https://go.dev/dl/

- Git
    - Winget: winget install --id Git.Git -e --source winget
    - Website: https://git-scm.com/install/windows

- Mpv
    - Winget: winget install --id shinchiro.mpv -e --source winget
    - Website: https://mpv.io/installation/

- yt-dlp
    - Winget: winget install yt-dlp
    - Webiste: https://github.com/yt-dlp/yt-dlp/releases and install latest .exe
